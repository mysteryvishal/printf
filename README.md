# printf

## Description

- `%c`         : ASCII character
- `%s`         : Text string (with `\0`)
- `%p`         : Memory address
- `%d` `%i`    : (Signed) Integer
- `%u`         : (Unsigned) Integer
- `%x` `%X`    : (Unsigned) Hexadecimal
- `%`          : Prints percent symbol
